# Enciclopedia de Contribución Open Source

Este repositorio reúne una guía técnica completa, estructurada por capítulos, orientada a entender, practicar y dominar el arte de contribuir a proyectos Open Source.  
El objetivo es entregar un conocimiento claro, práctico y accesible para estudiantes, desarrolladores, autodidactas y entusiastas del software libre.

## Contenido
- Explicaciones de conceptos fundamentales.
- Guía de herramientas.
- Ejemplos técnicos reales.
- Buenas prácticas.
- Microservicios, flujos de CI/CD, issues, PRs y más.

## Estructura del repositorio
- `/capitulos` – Contiene todos los capítulos del libro técnico.
- `/recursos` – Diagramas, ejemplos de código y material visual.
- `/licencia` – Licencia oficial para compartir el contenido.
